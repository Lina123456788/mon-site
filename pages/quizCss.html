1. A quoi sert le sélecteur CSS3 "h1 ~ p"?

    à cibler tous les paragraphes qui suivent un titre h1
    à cibler les éléments h1 s'ils contiennent unparagraphe
    à cibler tous les titres et paragraphes de la page
    à cibler tantôt un titre h1, tantôt un paragraphe

3.Dans quoi définit-on les déclarations CSS ?

    Entre deux crochets séparées par des tirets [ ..-.. ]
    Entre deux accolades séparés par des points-virgules { ... ; ... }
    Dans un fichier texte de type ".doc" ou "docx"
